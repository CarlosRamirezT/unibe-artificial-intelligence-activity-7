Sistemas de IA de éxito comercial

Describir dos sistemas desarrollados de IA y de éxito comercial que hayan sido creados empleando los lenguajes de programación seleccionados. (Franjul)

OpenAI Codex (Desarrollado con Python y JavaScript)

Descripción: OpenAI Codex es el modelo de IA detrás de GitHub Copilot. Se basa en GPT-3 y está diseñado para comprender y generar código en varios lenguajes de programación.

Lenguajes Utilizados: Python (para el desarrollo del modelo de IA y backend) y JavaScript (para la integración con el editor de código en GitHub Copilot).

Éxito Comercial: Ha sido ampliamente adoptado por desarrolladores para aumentar la productividad en la escritura de código, con GitHub Copilot generando ingresos significativos a través de suscripciones.

Unity ML-Agents (Desarrollado con C# y Python)

Descripción: Unity ML-Agents es un toolkit de aprendizaje automático que permite a los desarrolladores entrenar agentes inteligentes en entornos de Unity mediante técnicas como el refuerzo profundo.

Lenguajes Utilizados:

  • Python (para el entrenamiento de modelos de IA usando TensorFlow o PyTorch).
  • C# (para la integración con el motor de juegos Unity).

Éxito Comercial: Se ha utilizado en videojuegos, simulaciones industriales y en la investigación académica, convirtiéndose en una herramienta clave para el desarrollo de IA en entornos simulados.
